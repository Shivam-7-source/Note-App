# 📝 Notes App (Full Stack)

A simple notes application where users can create, update, and delete notes.

---

## 🚀 Features

* 📝 Create Notes
* 📖 View Notes
* ✏️ Update Notes
* 🗑️ Delete Notes

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Tailwind CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB

---

## 📁 Project Structure

backend/
client/

---

## ⚙️ Setup

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm start
```

---

## 🔗 API

| Method | Endpoint       | Description |
| ------ | -------------- | ----------- |
| GET    | /api/notes     | Get notes   |
| POST   | /api/notes     | Create note |
| PUT    | /api/notes/:id | Update note |
| DELETE | /api/notes/:id | Delete note |
